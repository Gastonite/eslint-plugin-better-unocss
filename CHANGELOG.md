# Changelog

## 0.1.0

Initial release.

### Features

- `better-unocss/order` - Sort classes using UnoCSS engine
- `better-unocss/no-duplicate-classes` - Remove duplicate classes
- `better-unocss/no-unnecessary-whitespace` - Normalize whitespace
- `better-unocss/enforce-line-wrapping` - Force multiline when classes exceed threshold

### Supported contexts

- Function calls: `cn()`, `clsx()`, `classnames()`, `twMerge()`, `cva()`
- Tagged template literals: `` cn`...` ``, `` tw`...` ``
